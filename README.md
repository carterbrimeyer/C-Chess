# C-Chess

Simple chess game written in C++ for COM S 327 2021 by Carter Brimeyer

=== HOW TO USE ===

-> open the directory in your terminal.

-> run 'make'.

-> run './chess' to launch game.

-> when done with game 'make clean' will clean up generated files.

===   ===   ===
